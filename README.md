# Uppröðun verkefnis
Verkefninu er raðað niður í ýmsar möppur.

## ./src/
Hérna eru geymdar allar skrar forritsins.

## ./design/
Hér eru upplýsingar um hönnun kerfisins og ýmis rit tengd því.

## ./docs/
Documentation um hvernig er hægt að nota forritið.
